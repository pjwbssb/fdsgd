第659章 实锤了，搞子就是歪嘴龙王！！！
搞子火了。
而且是以跟娱乐圈擦边的方式火的。
不管怎么说，
这一次都算是稍微蹭进去一点了。
而且是凭借他自己的实力！！
只不过……
现在人却消失了。
这就让大家感觉到很不正常。
公司在找林夏。
小赵队也在找林夏。
导演也在找林夏。
就连全网的网友……
也都在找林夏。
“啥情况？大型寻人纪录片，搞子去哪儿？”
“不造啊，昨天下午好像就没搞子的消息了。”
“嘶！这家伙不会又偷偷拍剧，准备惊艳我们所有人了吧？”
“还别说，你还真别说，搞子这歪嘴龙王还真是尬出新高度了。”
“兄弟们，我仿佛已经闻到了变革的气息，你们说有没有一种可能，娱乐圈风口要变了？”
“啊这？不至于吧？全都来玩尬的是吧？”
“尬不尬的先不说，你们就说你们爱不爱看就行了。”
“讲道理，这玩意儿确实上头，我承认，我是土狗，我爱看，哈哈哈哈！”
“所以，搞子到底去哪儿了？是不是去拍续集去啦？”
“你们说，有没有可能搞子是嘎了？”
“众网友：？？？？搞友来的是吧？”
“……”
网络上热闹的不行。
全是在打听林夏的话题。
毕竟，往常有人视频火了之后。
当事人立马就会跳出来疯狂推广。
可现在……
搞子这歪嘴龙王一夜席卷全网。
结果当事人竟然消失了？？
这不离大谱呢！
搞友们还都等着搞子开播，
疯狂弹幕攻击他呢。
……
找不到搞子。
搞友们只好开始搞事。
于是……
没过多久。
跟搞子有关的葬礼话题就开始上热搜了。
热巴看到这热搜之后，顿时笑喷了。
“噗哈哈哈！搞子又嘎一次，他这群搞友简直太离谱了！”
杨蜜跟赵若楠听完热巴的描述，嘴角也是忍不住抽了抽。
“这群人……还真是……”
赵若楠是早上来嘉行的。
因为早上没找到搞子，以为搞子来公司了。
所以她就直接来公司了。
只不过，来到公司之后发现。
搞子也不在公司。
而且杨老板他们也在找搞子。
三人合计了一番，心里顿时就有了想法。
这种情况下，
稿子不出来嘚瑟，反而玩消失。
八成是又去军方那边了。
但至于军方那边到底出了什么事。
她们就无从而知了。
不过，知道搞子没事儿。
她们也就不担心了。
杨蜜看完搞子的歪嘴龙王，
一眼就被赵若楠的形象吸引了。
所以……
现在的她在疯狂勾引赵若楠来嘉行。
“小楠楠~~真的，来我们嘉行吧，你绝对比搞子有前途！！”
“我保证，只要你来，立马就可以跟热巴姐妹花出道，你的形象真的太适合当艺人了！”
这话杨老板一点都没夸张，之前一起参加综艺节目的时候，她就觉得赵若楠形象不错。
现在看完短剧里的造型，更是被狠狠飒到了。
众所周知，杨老板最爱美的事物，包括…女人！
之前没邀请是因为赵若楠没兴趣。
现在既然都开始参与短剧拍摄了，
那再不赶紧拉到自己公司，后边可就来不及了！
赵若楠听完杨老板的邀请，笑了笑，“其实我真的没什么当艺人的想法，不过昨天拍短剧还挺好玩的，后边倒是可以考虑接触接触，反正我这边基本是离职状态，影响不大。”
赵若楠现在还是停职状态，但随时可以离职。
只不过，她这种人才，其实上边不一定愿意放手。
但话又说回来，刑警队这种地方，肯定是要依法办事的。
上次的事情闹那么大，他们又不能不处理赵若楠。
其实他们也不想的，但这都是上边的要求，他们又有什么办法呢？
而对于赵若楠来说。
现在其实当不当刑警都无所谓了。
当刑警也是不停的跟罪犯战斗。
现在不当刑警，其实也不会闲着。
依然在抓罪犯的路上。
所以，就她个人而言，是几乎没什么区别的。
但想到自己来了嘉行，就是搞子的同事了。
说不定还比搞子混的好？
赵若楠心里的那点恶趣味一下子就上来了。
“行！我考虑来你们公司，但是合同最好自由点，哪怕工资低点也行，主要我怕我做不好。”
杨蜜惊喜道：“放心吧，就按照搞子的合同来可以吧？以后咱都是一家人，什么做好做不好，绝对没问题的！！”
“嗯嗯，那我一会儿去办离职，我那边没什么问题。”
“好！我下午刚好没事儿，可以跟你一起去，我开车带你去呗？”
“也行！”
“走走走，热巴跟着一起，咱先出去吃顿饭庆祝一下，然后陪着小赵队去办离职！”
“今天开心，老板我请客！！”
一说到吃的，热巴自然是举双手赞同。
只是……
不知道为什么。
刚才听到说要跟搞子走一样的合同。
